# 🎵 MusicDB Manager

Aplicação para gerenciamento de banco de dados de músicas do aplicativo **Dark Spotify**, construída com **Electron** e **SQLite3**.  

---

## 🚀 Funcionalidades

- Adicionar músicas  
- Adicionar artistas  
- Adicionar álbuns  
- Exportar banco de dados como **JSON único**  
- Exportar banco de dados como **JSON por letras** (vários arquivos por letra)  
- Fazer **scraper** de músicas de sites  
- **Buscar músicas, artistas ou álbuns**  
- **Editar e remover** músicas, artistas ou álbuns  
- Visualização detalhada de cada música, artista ou álbum  

> 💡 Sugestões de funcionalidades futuras:
> - Suporte a **tags e gêneros musicais**  
> - Sistema de **favoritos**  
> - **Importação de CSV ou planilhas**  
> - **Relatórios estatísticos** (quantidade de músicas por artista, álbuns por gênero, etc.)  
> - **Interface gráfica avançada** com filtros e busca rápida

## 📝 Instalação e Uso

1. Baixe a versão mais recente  
2. Extraia o arquivo em uma pasta  
3. Abra o **MusicDB Manager.exe** (Windows) ou equivalente para macOS/Linux  
4. Comece a adicionar músicas, artistas e álbuns  

---

## 🛠️ Pré-requisitos

- Node.js (recomendado versão LTS)  
- Electron  
- SQLite3
